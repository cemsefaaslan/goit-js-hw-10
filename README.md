# goit-js-hw-10

GoIT JavaScript — Homework 10: Timers & Promises

## Tasks

- **Task 1** — Countdown Timer (`01-timer.html` / `js/01-timer.js`)
- **Task 2** — Promise Generator (`02-snackbar.html` / `js/02-snackbar.js`)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Libraries Used

- [flatpickr](https://flatpickr.js.org/) — date/time picker
- [iziToast](https://izitoast.marcelodolza.com/) — toast notifications
- [Vite](https://vitejs.dev/) — build tool
